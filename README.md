# CUI-utils
Character User Interface utilties for Python

## Install

```bash
       $ cd ~
       $ python3 -m venv env1
       $ cd env1
       $ source ./bin/activate

(env1) $ pip install -U pip setuptools wheel
       :
(env1) $ hasn -r

(env1) $ git clone https://github.com/ytani01/CuiLib.git
       :
(env1) $ cd CuiLib
(env1) $ pip install .
```
